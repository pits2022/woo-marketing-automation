# Internationalization (i18n) Fixes Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace remaining hardcoded strings in the WooCommerce Marketing Automation plugin with proper WordPress translation functions using the `woo-marketing-automation` text domain.

**Architecture:** We will modify the PHP classes `WMA_Admin` and `WMA_Activator` to wrap all untranslated strings in `__()`, `esc_html__()`, or `esc_attr__()` as appropriate. After modifying the PHP files, we will update the existing `.po` and `.mo` files in the `languages` directory using standard WordPress translation compilation commands (or a WP-CLI workflow, though we will focus on updating the source files first). We must also ensure the text is friendly and quiver-y where appropriate.

**Tech Stack:** PHP 8.1+, WordPress 6.9+

---
